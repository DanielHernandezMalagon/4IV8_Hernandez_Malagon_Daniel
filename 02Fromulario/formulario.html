Todos los formularios que estan en internet tienes metodos de programacion por defecto.Estos metodos nos sirven para poder hacer las acciones principales como son: Acciones para poders conectar con los servidores, a traves de las Arquitecturas cliente servidor. Es que vamos a tener por definicion que todos los formularios utilizan el metodo GET, y el segundo metodo principal es POST, aunque existen otros metodos que se encargan del intercambio de información. GET: este metodo envia por medio de la URL toda la informacion del usuario a traves del navegador. Su función principal es obtener los datos del servicio; pero eds el metodo mas vulnerable de todos derivados que se pueden generar inyecciones de codigo. POST: este metodo se encarga de registrar una nueva entrada al sistema tambien utiliza la URL como envio por defecto; pero su facultad es poder ocultarla, esto significa que los datos no son visibles para el usuario comun en el navegador.
